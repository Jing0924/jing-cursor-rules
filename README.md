# Jing's Cursor Rules

[中文版](./README.zh-TW.md) | English

A personal collection of [Cursor](https://www.cursor.com/) Rules organized by technology category, making AI collaboration more consistent and efficient.

## Usage

Copy the desired `.mdc` files into your project's `.cursor/rules/` directory:

```bash
cp commit/commit-rules.mdc your-project/.cursor/rules/
cp nextjs/nextjs.mdc your-project/.cursor/rules/
```

## Directory Structure

```
jing-cursor-rules/
├── commit/          # Git commit conventions
└── nextjs/          # Next.js related rules
```
